recordError

fun PendingTask.recordError(humanReadableErrorMessage: String?, errorId: String?): PendingTaskResult

Extension to Wendy.recordError easily from a PendingTask instance.

Return
PendingTaskResult.FAILED The extension automatically returns PendingTaskResult.FAILED for you so that you can simply call: return recordError() in your PendingTask to avoid return PendingTaskResult.SUCCESSFUL by accident from your PendingTask.
